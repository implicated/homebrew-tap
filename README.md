# Implicated Tap

## How do I install these formulae?

`brew install implicated/tap/<formula>`

Or `brew tap implicated/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "implicated/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
